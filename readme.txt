Install
-------
	1) create directory 'bin' in your home directory
	2) add ' export PATH=$PATH":$HOME/bin" ' to your .bash_profile
	3) refresh your bashprofile... ' source .bash_profile '
	3) cd to /home/ugrads/meade/shared
	4) cp chat ~/bin
	5) cp checkChat ~/bin
	

How To Use
---------
	1) Recommend: open 2 window sessions
	2) enter 'checkChat' in cmd line
	3) go to second window, enter 'chat -con' in cmd line 
			[Examples]
			chat -con  // this gives a continous chat connection
			chat "Long message" // this is used for long messages with a one time send
			chat Yo // This is used for a short one word message with a one time send
